ScratchJrBlocks

Enter the name of a ScratchJr block (e.g. forward) to create an SVG version of it! Enter several in a row (e.g. flag right forever) to create a script. Press enter (break to a new line) to create more than one script. Enter ellipses (...) or general names (e.g. block) to create placeholder blocks.

Not all blocks have been added yet. This is still in a very work-in-progress stage. Progress will be made very soon! This early version was released to show what's been done so far.
